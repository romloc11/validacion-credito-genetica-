# Automatización de Validación de Antigüedad de Crédito

Script en Python que integra información desde múltiples fuentes (SAP, CRM y límites de crédito)
para identificar clientes elegibles para el sistema Genética.

## Problema
El proceso se realizaba de forma semi-manual con macros y fórmulas de Excel,
lo que generaba errores y consumía varias horas de trabajo.

## Solución
- Integración automática de 3 bases de datos
- Normalización de códigos SAP y fechas
- Cálculo de meses de antigüedad
- Clasificación automática de clientes

## Tecnologías
- Python
- Pandas
- NumPy
- Excel
